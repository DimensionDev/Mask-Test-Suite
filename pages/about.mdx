# About

This website is dedicated to the collection and presentation of in-development test suites for [Mask Network](https://mask.io) extension.

To get started, simply browse our collection of test suites, which are organized by category. You can also use our search function to find tests by keyword.

Here are some useful links:

-   https://nextra.site/ The offcial website of nextra which is the generator of this website.
-   https://commonmark.org/ The Markdown syntax we are using to write test cases.
-   https://www.lambdatest.com/learning-hub/test-suite An arbitrary document I found talks about writing test suites.

If you have any questions or feedback, please don't hesitate to contact us.
